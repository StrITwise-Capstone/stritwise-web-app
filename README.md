## Folder Structure

After creation, your project should look like this:

```
stritwise-web-app/
  node_modules/
  package.json
  public/
    assets/
    index.html
  src/
    components/
      Error/
      UI/
    config/
    hoc/
      Layout/
    routes/
      some-route/
      routes.js
    store/
      actions/
        actionTypes.js
        index.js
        someAction.js
      reducers/
        someReducer.js
    App.js
    App.test.js
    index.css
    index.js
  .eslintrc
  .gitignore
```
